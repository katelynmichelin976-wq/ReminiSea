# 忆海拾光 · 用户手册

> 版本：v5.11.2（2026-06-09）

---

## 一、这个 App 是什么

**忆海拾光** 是一款家庭记忆与学习卡片 App，把家人面孔、生活场景、文字词汇做成卡片，每天花十几分钟练习，借助间隔重复的方法把记忆维持得长久。

- **离线即可使用**——卡片导入手机后，全程无需联网
- **登录后多设备同步**——同一账号在手机和 iPad 上看到相同的进度
- **支持图片、语音、文字三种卡片**
- **不上传任何敏感信息**——只同步学习记录与你导入的卡片内容

打开网址即可使用：https://katelynmichelin976-wq.github.io/ReminiSea/

建议添加到主屏幕（手机浏览器的「分享 → 添加到主屏幕」），用起来跟原生 App 一样。

## 二、第一次使用

### 1. 打开 App

第一次打开会看到首页，顶部是「忆海拾光」标题，底部三个按钮：**首页 / 开始练习 / 我的**。

中间一片空白，因为还没有任何牌组。先去导入一份。

### 2. 导入第一份牌组

牌组以 `.yhspack` 文件的形式打包好（朋友或家人发给你的链接、网盘文件）。

**步骤：**
1. 把 `.yhspack` 文件保存到手机
2. 在文件管理里点这个文件
3. 选择「用忆海拾光打开」（或在 App 内通过设置 → 导入）

导入成功后，首页会出现这副牌组的卡片，红色背景表示当前选中。

### 3. 开始练习

点底部「**开始练习**」按钮，跳出题目界面，选项排在下方。

- **点对的选项**：变绿，下一张
- **点错的选项**：错选项淡出，再选一次（轻松模式可关掉这个重试）
- **完成所有题目**：自动跳到「完成」屏，显示这一局的统计

## 三、登录与多设备同步

不登录也能完整使用所有学习功能；登录的好处是**多设备同步**。

### 1. 注册账号

底部「**我的 → 点击登录**」，进入账户屏。

- 第一次：在邮箱栏填邮箱、密码栏填密码（至少 6 位），点「登录」
  - 如果账号还不存在，会自动创建
- 之后：直接填同一个邮箱密码即可
- 邮箱建议用真实邮箱（找回密码时用得到）

### 2. 多设备使用

在另一台设备（如 iPad）上打开同一网址，在「我的 → 点击登录」处填**相同**的邮箱和密码，App 会自动把已有的牌组、学习记录拉下来。

**注意：** 同步要 1-2 分钟才能完成，下载图片的话更久。耐心等到进度结束再练习。

### 3. 退出登录

「我的 → 账户 → 退出登录」。本地的牌组与练习记录会保留，但不再与云端同步。

## 四、首页

首页是你练习的入口。

### 牌组列表

- 每一行是一副牌组，显示**牌组名 / 待复习张数 / 新卡片张数**
- 点击牌组：把它选为「当前牌组」（红色背景，左侧出现红条）
- 选好之后点底部「开始练习」就练这副

### 右上角

- **版本号**：连续点 5 次会打开技术诊断面板（一般用不到）
- **刷新按钮**：强制刷新数据，遇到显示异常时用

### 黄点（待同步标记）

牌组名后面有时会出现一个**橙色小圆点**，表示这副牌组的内容或学习记录与云端不一致，需要同步。

- 点击黄点会跳到云端同步页（高级模式下）
- 普通使用下：放着不管，下次自动同步

## 五、练习模式

练习模式决定每一局怎么出牌。在「**我的 → 设置 → 练习模式**」切换。

### 普通模式（normal）

完整的间隔重复算法。

- 出所有「到期复习」的卡片
- 按到期时间从早到晚排
- 答对：间隔加长
- 答错：跌回学习中状态，间隔重置
- 适合：每天稳定练习的用户

### 轻松模式（easy）

陪伴式练习，更照顾情绪。

- 每局**固定 15 / 19 / 23 张**（可在设置中选）
- 结构：开头 3 张熟悉卡热身 → 中间穿插「3 熟 + 1 挑战」节奏 → 末尾几张熟悉卡收尾
- 答错不降级，不会把卡片打回学习中
- 适合：陪老人练习、希望低压力的用户

#### 轻松模式的「答错重试」开关

- **打开（默认）**：答错后错选项淡出，可以再选一次
- **关闭**：答错直接揭晓正确答案，进入下一张

数据记录是一样的——无论开关位置，App 都把「第一次答错」记下来，用于评估这张卡的熟悉度。

## 六、牌组管理（高级模式）

普通模式下，首页只能选牌组练习，无法重命名/删除。打开**高级模式**后底部会多出「**牌组**」和「**统计**」两个按钮，能进入牌组管理页。

### 1. 怎么开高级模式

「我的 → 高级模式」开关打开。

### 2. 牌组管理页（screen-decks）

底部点「**牌组**」，进入三段式管理：

- **本地**：你这台设备已有的牌组，左滑可以**导出 / 重命名 / 删除**
- **云端**：登录后才显示。云端存的所有牌组列表
  - 「下载」：把还没在本设备的牌组拉下来
  - 「同步」：本地与云端的内容对齐
  - 「补全媒体」：媒体还没下载完时点这个
- **精选**：占位，后续上线官方牌组

### 3. 同步状态徽章

云端牌组每一行右边会显示状态：

| 徽章 | 含义 |
|---|---|
| 已同步 | 本地与云端完全一致 |
| 待上传 N | 本地有 N 张卡的修改/新增等待上传 |
| 待下载 | 云端有更新，需要拉下来 |
| 双向 +N | 两边都改了，N 张待处理 |
| 媒体缺失 | 卡片信息已下载，但图片/语音文件还没拉完 |

### 4. 暂停 / 继续下载

下载大牌组时按「暂停」可以中断，下次进来按「继续」从上次断点续传。流量紧张或要换 WiFi 时用得上。

## 七、统计页

底部「**统计**」按钮（高级模式可见）打开统计面板，显示：

- 今天复习 / 答对 / 答错的张数
- 历史柱状图：最近若干天的练习量
- 各牌组的累计统计

进入统计后右上角有返回按钮，回到上一屏。

---

## 八、我的页

底部「**我的**」进入个人页，提供：

### 顶部账户卡片

显示当前登录邮箱（或「点击登录」字样）。点击进入账户屏。

### 菜单项

- **设置**：练习参数（练习模式、题数、答错重试、语音、答案朗读、答题超时等）
- **主题**：浅色 / 深色 / 跟随系统
- **语言**：中文（简体 / 繁体）、English、Español、日本語
- **关于**：App 信息、版本、隐私说明
- **意见反馈**：直接发送意见到开发者，可附带诊断信息（仅本地数据，无任何敏感信息）

### 高级模式开关

最下方的「⚡ 高级模式」开关，打开后底部 Tab Bar 多出「牌组」「统计」两项，并解锁牌组管理页的左滑操作。

## 九、设置详解

「我的 → 设置」中常用项：

### 练习模式

- 普通 / 轻松 二选一（详见第五节）

### 题数（轻松模式专属）

- 15 / 19 / 23 三档单选
- 19 张约 9-10 分钟，是默认推荐
- 15 适合时间不多、注意力短的日子
- 23 适合状态好想多练

### 答错重试（轻松模式专属）

- 开 / 关，影响错题后是否给重试机会（详见第五节）

### 语音

- **答案朗读**：揭晓答案时是否朗读卡片名称
- **答题超时**：每张卡片可以停留多久（默认无超时）
- **音色**：选择 TTS 朗读音色
  - 自动：跟随界面语言
  - 也可手动指定具体声音（如普通话 / 粤语 / 英语男声）

### 进入语音辅助管理

点击「语音辅助管理」进入全屏管理页，可以：

- **录制家人的录音**填充各情绪触发槽位（如「答对鼓励」「答错安慰」「连对庆祝」）
- **覆盖默认 TTS 文案**（每个槽位都有默认中文文案，可改）

共 11 个语音槽：
- 固定节点：开始、结束、提示
- 情绪触发：答对、答错、连对 3 次
- 功能提示：浏览引导、休息提醒等

录制的录音存在本设备，**会跟着账号同步到其他设备**（按住到「家庭录音」槽位选择文件上传）。

## 十、语言

「我的 → 语言」一键切换界面语言。当前支持：

- **English**
- **中文（繁体）**
- **中文（简体）**
- **日本語**
- **Español**

切换语言会同时改变 TTS 默认音色（如切到日文，朗读自动用日文声）。如果你希望保留中文音色但用日文界面，可以在「设置 → 音色」手动锁定。

## 十二、常见问题

### 同步失败怎么办？

1. 检查网络
2. 确认右上角刷新一次
3. 如果某副牌组卡在「同步中」很久，进牌组管理页「云端 Tab」按「暂停」再「继续」
4. 仍失败：截图通过「意见反馈」发给开发者

### 牌组名旁边的黄点一直不消失？

通常是有未同步内容。手动进入云端 Tab 点「同步」即可。

### 换了新设备，旧设备的数据不见了？

新设备登录后**等 1-2 分钟**让 App 拉数据。第一次同步会下载所有牌组与学习记录，没拉完之前看到的是空白属于正常现象。

### 离线练习后再登录，进度消失了？

**已知问题。** 如果你长时间离线（数天）后登录，本地的练习进度可能不会上传到云端。建议**先登录，再练习**。已 fix 计划中。

### 卡片图片显示不出来？

牌组管理页云端 Tab 看是否有「媒体缺失」徽章，点「补全媒体」会重新下载。

### 「高级模式」是什么？

进阶用户用的模式，开启后底部多出「牌组」「统计」两个 Tab，并解锁牌组左滑编辑。普通使用不必打开。

### 怎么删除一副牌组？

需要开启高级模式 → 底部「牌组」→ 本地 Tab → 左滑要删的牌组 → 「删除」。

### 怎么导出牌组分享给别人？

高级模式 → 底部「牌组」→ 本地 Tab → 左滑 → 「导出」，得到 `.yhspack` 文件，发给对方即可。

### 数据安全吗？

- **本地数据**：保存在浏览器的 IndexedDB 中，不离开你的设备
- **云端数据**（登录后）：仅同步**卡片内容、练习记录、自定义语音**，全部存在 Supabase（加密传输、加密存储），不向任何第三方共享
- 你随时可以删除账号并清空所有云端数据（联系开发者）

- 顶部「忆海拾光」右边显示当前版本（如 `v5.11.2`）
- 连续点 5 次版本号会打开诊断面板，开发者排查问题时用

## 十四、意见反馈

「我的 → 意见反馈」打开反馈表单，写下你的问题或建议，可勾选「附上诊断数据」帮助排查。诊断数据不含任何敏感信息（不上传邮箱/密码/手机号）。

发出后通常 1-3 天内会得到回复。

## 十五、词汇表

| 词 | 解释 |
|---|---|
| 牌组（deck） | 一组主题卡片的集合，比如「家人面孔」「常用词汇」 |
| 卡片（card） | 牌组中的一张题目，包含问题、答案、可选的图片/音频 |
| 同步 | 把本地内容与云端对齐的过程 |
| 高级模式 | 给进阶用户的功能层，普通用户不必开启 |
| TTS | Text-to-Speech，App 自动朗读文字的功能 |
| `.yhspack` | 忆海拾光的牌组打包格式（一个文件里包含卡片信息 + 媒体） |
