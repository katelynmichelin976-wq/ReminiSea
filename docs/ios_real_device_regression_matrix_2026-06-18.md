# iOS 真机回归矩阵

日期：2026-06-18

范围：手工测试清单，不改代码。用于 iOS Safari、主屏 PWA、Capacitor/WKWebView/TestFlight 三种运行环境。

## 设备矩阵

| 设备档位 | 最低覆盖 | 重点风险 |
| --- | --- | --- |
| 小屏 | iPhone SE 或等价 375px 宽设备 | 按钮过小、文字截断、软键盘遮挡、底部操作被 Home Indicator 遮挡 |
| 标准屏 | 当前主流 iPhone | 默认体验、常规性能、safe area |
| 大屏 | iPhone Pro Max | 卡片尺度、长文本行宽、底部导航可达性 |
| iPad 可选 | iPad portrait | 宽屏阅读长度、固定底栏宽度、sheet 尺寸 |

## 运行环境

| 环境 | 必测 | 说明 |
| --- | --- | --- |
| iOS Safari | 是 | 验证普通 Web 访问 |
| 主屏 PWA | 是 | 验证 standalone、safe area、状态栏、返回行为 |
| WKWebView/TestFlight | 是 | 验证 Capacitor 包、权限弹窗、IndexedDB 持久化 |
